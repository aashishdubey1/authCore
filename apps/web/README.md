# Web App

Lightweight Next.js frontend for AuthCore backend.

## Features
- Register
- Verify email
- Resend verification
- Login
- Logout
- Logout all sessions
- Meme messages for guest/logged-in users
- Quest board that nudges users to try every auth route

## Run
```bash
npm install
npm run dev:web
```

By default API base is `http://localhost:4000` (configurable with `NEXT_PUBLIC_API_BASE_URL`).
