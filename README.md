# authCore Monorepo

This repository is organized as:
- `apps/api`: Bun + Express + Prisma auth backend
- `apps/web`: Next.js frontend with fun auth quest UX

## Quick Start
1. Install dependencies at root:
```bash
npm install
```
2. Start API:
```bash
npm run dev:api
```
3. Start web app:
```bash
npm run dev:web
```
